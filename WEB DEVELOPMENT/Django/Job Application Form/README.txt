### Set-up the project

## 1) Go to the directory where is the script using the terminal
## 2) Write on anaconda prompt: django-admin startproject mysite .

### Create the app
## 3) Write on anaconda prompt: python manage.py startapp job_application

### Run the app
## 4) Run on anaconda prompt: python manage.py runserver

### Make migrations to transfer code automatically
## 5) python manage.py makemigrations
## 6) python manage.py migrate

### When creating the Admin Interface
## 7) Write on anaconda prompt: python manage.py createsuperuser
## 8) To enter to the admin web, write on the internet search bar: http://127.0.0.1:8000/admin

### When creating the About Us page
## 9) To enter to the Abput Us page, write on the internet search bar: http://127.0.0.1:8000/about
